Why not use PID control?

• We could, but:

• The gains for PID are good for a small region of state-space.

• System reaches a state outside this set → becomes unstable • PID has no formal guarantees on the size of the set

• We would need to tune PID gains for every control variable.

• If the state vector has multiple dimensions it becomes harder to tune every control variable

in isolation. Need to consider interactions and correlations.

• We would need to tune PID gains for different regions of the state-space and

guarantee smooth gain transitions

• This is called gain scheduling, and it takes a lot of effort and time
